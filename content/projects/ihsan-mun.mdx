## 🚀 Project Overview

**Ihsan MUN Landing Page** is a vibrant, modern website created to promote the Model United Nations event organized by Ihsan International Schools. Targeted towards students and international participants, the site focuses on a clean, responsive user experience that inspires confidence and drives engagement.

## 🎯 The Challenge

The project aimed to solve these core needs:

- Create an official online presence for the Ihsan MUN event.
- Design a professional and trustworthy landing page to represent the school internationally.
- Make the website fast, responsive, and visually engaging to encourage registration.

## 💡 Our Solution

I crafted a sleek landing page combining modern UI principles with interactive elements:

- Fully responsive, mobile-first design.
- Elegant UI components using **shadcn/ui** and custom TailwindCSS utilities.
- Smooth and expressive animations powered by **Motion.dev** for a dynamic feel.
- SEO optimization and performance best practices for fast loading.

## 🛠️ Technical Architecture

- **Frontend:** Next.js (App Router)
- **Styling:** TailwindCSS + shadcn/ui
- **Animations:** Motion.dev
- **Hosting:** Vercel (instant deployments and optimizations)

## 🏆 Key Achievements

- ✅ Successfully launched and used as the official site for Ihsan MUN.
- ✅ Delivered within tight timelines before the event launch.
- ✅ Positive internal feedback from school administrators and MUN organizers.
