## 🚀 Project Overview

The **ESP32 Drone Project** is a lightweight, low-cost drone built around the **ESP32-WROOM-32E** microcontroller. It features real-time data processing and communication between the drone and a mobile app via Wi-Fi.

The system is designed to support basic drone functions such as:

- Altitude measurement using a **BMP280** sensor.
- Orientation and motion sensing using an **MPU6500** IMU.
- Motor control via **DRV8833** and **coreless motors**.
- Remote control and data visualization through a **React Native app**.

---

## 🔧 Hardware Components

- **ESP32-WROOM-32E** – The brain of the drone.
- **MPU6500** – IMU sensor for orientation and motion sensing (SPI).
- **BMP280** – Barometric pressure sensor for altitude estimation.
- **DRV8833** – Dual H-Bridge motor driver to control coreless motors.
- **Coreless Motors (7x20mm)** with **55mm Propellers**.
- **3.0V Li-Po Battery** for power supply.

---

## 📱 Software Components

- **ESP-IDF** – For firmware development on the ESP32.
- **React Native + Expo** – For building the mobile control app.
- **Wi-Fi Station Mode** – The drone hosts a local web server for real-time communication.
