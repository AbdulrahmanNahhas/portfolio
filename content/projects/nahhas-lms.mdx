## 🚀 Project Overview

**Nahhas LMS** is a modern, open-source Learning Management System (LMS) crafted with **Next.js 14**. Designed for online education, it provides a flexible platform for course creation, purchase, and student tracking. The project was developed as a personal learning challenge and has since been open-sourced for the benefit of others.

> 🕰️ _Note: This project is archived and hasn’t received updates since July 2024. Contributions are welcome!_

## ✨ Key Features

- ✅ Browse and filter courses
- 💳 Purchase functionality powered by **Stripe**
- 🧠 Track chapter and course progress
- 👨‍🎓 Student dashboard for content access
- 👩‍🏫 Teacher tools for creating and managing courses
- 🧩 Drag-and-drop chapter ordering
- 🖼️ Upload videos and media via **UploadThing**
- 🎥 Video playback with `react-player`
- 📝 Rich text editing for course content
- 🔐 User auth via **Clerk**
- 📦 Data managed through **MongoDB + Prisma**

## 🧱 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **UI Libraries:** Tailwind CSS, Shadcn/UI, Radix UI
- **Database:** MongoDB with Prisma ORM
- **Authentication:** Clerk
- **Payments:** Stripe
- **Uploads:** UploadThing, Cloudinary
- **Forms:** React Hook Form
- **Markdown Editing:** React Markdown Preview & React MDE
- **API Layer:** Axios

## 🧪 Project Usage

Once installed and set up, you can:

- View available courses
- Log in as a teacher or student
- Create, manage, and track learning content
- Process payments securely
- Use the dashboard for real-time progress
