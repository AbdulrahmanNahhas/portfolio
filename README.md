# 🚀 Portfolio 2026

A high-performance personal portfolio website designed to showcase projects and technical expertise. Built with a focus on speed, localization, and modern UI patterns.

## 🛠️ Tech Stack

| Category | Tools |
| --- | --- |
| **Framework** | **Next.js 16** (App Router) |
| **Styling** | **Tailwind CSS v4**, **Shadcn/ui**, **Base UI** |
| **Content** | **MDX** & **Gray-matter** (Markdown-based projects pages) |
| **Icons** | **Phosphor Icons** |
| **Language** | **TypeScript**, **Next-intl** (i18n Support) |
| **Linting** | **ESLint** |

## ✨ Key Features

* **Next.js 16:** Leveraging the latest React Server Components and optimized rendering.
* **Tailwind v4:** Using the new engine for faster builds and modern CSS capabilities.
* **Internationalization:** Fully localized content via `next-intl`.
* **MDX Integration:** Seamlessly write project descriptions and blog posts in Markdown.
* **Accessible UI:** Built with **Base UI** and **Shadcn** for high accessibility (A11y) standards.

## 🚀 Getting Started

Follow these steps to get the project running locally:

### 1. Clone the repository

```bash
git clone https://github.com/AbdulrahmanNahhas/portfolio.git
cd portfolio

```

### 2. Install dependencies

```bash
npm install
# or
pnpm install

```

### 3. Run the development server

```bash
npm run dev
# or
pnpm dev

```

### 4. View the project

Open [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) in your browser.
